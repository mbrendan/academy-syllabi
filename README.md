# Cosmos Institute — Syllabi & Course Materials

A living collection of seminar syllabi, reading lists, and course structures developed through Cosmos Institute programs.

## Overview

Cosmos Institute has run seminars across multiple formats and venues, testing different approaches to the philosopher-builder curriculum. This repo contains the full reading lists, discussion questions, and facilitator notes from these programs.

**Core themes across all seminars:**
- Human autonomy in the age of AI
- Truth-seeking and epistemic institutions
- Decentralization and distributed systems
- The intellectual origins of technology
- The philosopher-builder
